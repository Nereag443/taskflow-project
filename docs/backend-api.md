AXIOS
Axios es una librería HTTP para node.js y el navegador.
Uso: 

POSTMAN
Postman es una plataforma de colaboración para el desarrollo de API, la cual permite probar endpoints, enviar request HTTP, guardar colecciones, automatizar test de API, y mucho más.

SENTRY
Sentry es una plataforma de monitoreo de errores en producción que permite detectar errores en tiempo real, ver stack traces, monitorear rendimiento, alertas automáticas, etc.

SWAGGER
Swagger es una herramineta para diseñar, probar y documentar API REST.
